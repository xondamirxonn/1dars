Edit Layout Grid

Quickly update multiple layouts by editing one Grid Style.

Select one of the frames and open the Grid Style menu. Then choose the “Edit Style” icon next to the selected style.

Update the margin from “24” to “40” for both the row and the column. Watch all the frames update in unison.
